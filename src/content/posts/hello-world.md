## 为什么建这个博客

大一这一年学到了很多东西，零零散散记在 Notion、Obsidian、甚至备忘录里。时间长了，找起来很费劲，而且写得随意，没有整理。于是就想，不如自己搭一个网站：

- **文章** 板块用来总结学过的技术
- **项目** 板块把我写的小玩具都整理出来，方便面试/找实习时展示
- **文档** 板块变成一个随用随取的知识库

## 技术选型

这个网站用 **Astro** 构建。选它的原因很简单——我想练手前端，但又不想上来就学 React/Vue 的复杂生态。Astro 对你友好：它支持 Markdown 写内容，可以内嵌组件，默认输出纯静态 HTML，性能很顶。

整个网站的工作流是：

```text
本地写 Markdown → git push → GitHub Actions 自动构建 → 部署到服务器 Nginx
```

一条命令就能更新全站，非常顺手。

## 接下来

慢慢把以前记的笔记搬过来，整理成像样的文章。给自己立个 flag：**一周写一篇**，哪怕短，也要坚持。

如果你在看我，谢谢你。这个空间会一直长下去 🌱
