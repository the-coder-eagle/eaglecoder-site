C Learn 是一个面向零基础的 C 语言学习平台，核心痛点：**不用装编译器**。

## 功能

- ⚡ 交互式终端（CLion 风格 I/O）
- 🎨 实时语法高亮（8 种 Token 类型）
- 📖 36 课系统化课程（从 Hello World 到链表）
- 💻 144 道在线判题（AC/WA/CE）
- 🔍 代码可视化模拟器（逐行执行 + 内存观察）
